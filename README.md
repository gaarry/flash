# 3D 粒子系统 - 手势控制

基于 Three.js 和 MediaPipe 的实时交互 3D 粒子系统。通过摄像头检测手势张合来控制粒子群的缩放与扩散。

## ✨ 功能特性

- 🎨 **多种粒子模型**: 爱心、花朵、星星、佛像、烟花、银河
- 🖐️ **手势控制**: 通过摄像头实时检测手掌张合程度，控制粒子扩散
- 🎨 **颜色自定义**: 提供颜色选择器和预设颜色方案
- ⚡ **实时响应**: 粒子实时响应手势变化，流畅动画
- 🖥️ **全屏支持**: 一键全屏沉浸式体验
- 📊 **性能监控**: 实时显示 FPS 和粒子数量

## 🚀 快速开始

### 方式一：直接打开

双击 `index.html` 在浏览器中打开（推荐使用 Chrome/Edge）。

### 方式二：本地服务器

```bash
# 使用 Python
python -m http.server 8080

# 或使用 Node.js
npx serve .

# 然后访问 http://localhost:8080
```

## 🎮 操作说明

### 手势控制

1. 点击右上角 📷 按钮开启摄像头
2. 将手掌伸向摄像头
3. **张开手掌** → 粒子扩散
4. **握紧拳头** → 粒子收缩

### 界面控制

- **模型选择**: 左侧面板选择不同形状
- **颜色调整**: 使用颜色选择器或预设颜色
- **粒子数量**: 滑块调整粒子密度（1000-15000）
- **粒子大小**: 滑块调整粒子尺寸
- **鼠标拖动**: 移动鼠标旋转视角
- **滚轮缩放**: 调整观察距离
- **全屏**: 点击 ⛶ 按钮
- **隐藏UI**: 点击 👁️ 按钮

## 🛠️ 技术栈

- **Three.js** - 3D 图形渲染
- **MediaPipe Hands** - 手势检测
- **GLSL Shaders** - 自定义粒子着色器
- **CSS3** - 现代玻璃态 UI
